# Order Summary Card

A clean and responsive order summary component built with HTML and CSS. This project was focused on mastering layout techniques and achieving a high level of design fidelity.

## 🧠 Learning Objectives
This project was part of my early journey in web development. The main goals were:
* Implementing a mobile-first responsive design.
* Working with **Flexbox** for alignment and centering.
* Styling components with custom fonts and hover states to improve UX.
* Understanding the CSS Box Model and background image scaling.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, Flexbox, and Media Queries.
* **Google Fonts:** "Open Sans" for typography.

## 📱 Features
* Fully responsive layout for mobile and desktop.
* Interactive elements with hover effects on buttons and links.
* Clean and maintainable CSS structure.
